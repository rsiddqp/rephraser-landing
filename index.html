Rephraser

Rephrase any text, in any app, instantly.

Download for macOS

Free • macOS 10.15+ • 8.8 MB

Setup

  1. 1 Unzip and move Rephraser.app to Applications
  2. 2 Open it and grant Accessibility permissions when prompted
  3. 3 Select any text anywhere, press Cmd+Shift+R

That's it. No account, no API key, no configuration.

How it works

You write:

hey can we push this to next week im swamped

Rephraser returns:

Would it be possible to reschedule this for next week? I'm currently managing several urgent priorities.

Choose from Professional, Casual, or Sarcasm styles. Press Cmd+C to copy and dismiss.

Advanced: Use your own AI

Rephraser works out of the box. If you prefer, you can bring your own API key from any of these providers:

OpenAI Anthropic Claude Google Gemini Perplexity

Configure in Settings inside the app. Keys are stored securely in your system keychain.
